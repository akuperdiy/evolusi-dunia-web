⚡ JavaScript: Bahasa Interaktivitas Web

  • Pencipta: Brendan Eich
  • Perusahaan: Netscape
  • Nama awal: Mocha → LiveScript → JavaScript

JavaScript diciptakan dalam waktu singkat untuk menghadirkan interaktivitas di browser. Meski awalnya sederhana, JavaScript berkembang menjadi bahasa pemrograman lengkap yang berjalan di mana-mana—client hingga server.

Standarisasi melalui ECMAScript memastikan JavaScript konsisten di berbagai browser dan membuka jalan bagi inovasi modern.

Fitur Awal

  • Manipulasi DOM sederhana, validasi formulir, dan event handling dasar.
  • Tipe dinamis dan fungsi sebagai warga negara kelas satu.

Dampak

JavaScript mengubah web dari halaman statis menjadi aplikasi interaktif. Ekosistem modern (framework, bundler, npm) bertumpu pada fondasi ini.
