⚡ JavaScript: Bahasa Interaktivitas Web

  • Pencipta: Brendan Eich
  • Perusahaan: Netscape
  • Nama awal: Mocha → LiveScript → JavaScript

JavaScript diciptakan dalam waktu singkat untuk menghadirkan interaktivitas di browser. Meski awalnya sederhana, JavaScript berkembang menjadi bahasa pemrograman lengkap yang berjalan di mana-mana—client hingga server. Evolusi selanjutnya distandarkan melalui ECMAScript oleh TC39 sehingga perilaku lintas-browser menjadi konsisten.

Standarisasi melalui ECMAScript memastikan JavaScript konsisten di berbagai browser dan membuka jalan bagi inovasi modern.

Fitur Awal

  • Manipulasi DOM sederhana, validasi formulir, dan event handling dasar.
  • Tipe dinamis dan fungsi sebagai warga negara kelas satu.

Ekosistem & Dampak

  • DOM APIs dan standar ECMAScript mematangkan pemrograman di browser.
  • Kelahiran Ajax (2005) dan SPAs mendorong pola aplikasi kaya di browser.
  • Dengan Node.js (2009), JavaScript merambah server; ekosistem npm tumbuh pesat.

Garis Waktu Singkat

  • 1995: JavaScript dirilis di Netscape.
  • 1997: ECMAScript edisi pertama diterbitkan (ECMA-262).
  • 2015: ES6 membawa lompatan fitur bahasa modern.

Glosarium

  • ECMAScript: Standar resmi bahasa yang diimplementasikan sebagai JavaScript.
  • TC39: Komite teknis yang mengembangkan standar ECMAScript.

Praktik Terbaik (Modern)

  • Gunakan module (ESM) untuk struktur kode yang terkelola.
  • Manfaatkan linting dan type checking (mis. TypeScript) untuk reliabilitas.
  • Utamakan async/await di atas callback-pyramid untuk asinkron yang lebih jelas.

Miskonsepsi Umum

  • "JS itu sama dengan Java": Keduanya berbeda; kemiripan nama adalah strategi pemasaran era Netscape.
  • "JS hanya untuk browser": Kini JS dipakai di server, desktop, mobile, hingga IoT.
