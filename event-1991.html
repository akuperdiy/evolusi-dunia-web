  • Organisasi: CERN
  • Perilisan: Agustus 1991 (pengumuman di alt.hypertext)
  • Komponen: Server info.cern.ch, browser WorldWideWeb/Nexus

Pada 1991, Tim Berners-Lee dan tim di CERN membuka akses World Wide Web ke komunitas internet global. Server pertama (info.cern.ch) dan perangkat lunak pendukung seperti WorldWideWeb (kemudian Nexus) memungkinkan siapa pun menjelajah dan menerbitkan dokumen terhubung melalui URL, HTTP, dan HTML.

Inilah momen ketika Web beralih dari eksperimen internal menjadi infrastruktur publik yang siap diadopsi luas.

Teknologi dan Praktik Awal

  • Server web httpd berjalan di mesin NeXT di CERN.
  • Browser/editor Nexus mendukung penulisan dan penelusuran dokumen.
  • Hypertext memungkinkan navigasi bebas antardokumen lewat tautan.

Dampak

Perilisan publik mempercepat kolaborasi riset, memicu munculnya server dan situs baru di universitas hingga industri, dan menjadi fondasi ekonomi digital di dekade berikutnya.

Garis Waktu Singkat

  • 1991: Web diumumkan ke publik; server dan browser tersedia.
  • 1992: Pertumbuhan server web di luar CERN.
  • 1993: Mosaic memopulerkan web grafis.
