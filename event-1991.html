Pengertian

Rilis Publik Web (1991) adalah saat ketika perangkat lunak dan dokumentasi Web‚Äîserver, browser, serta format dokumen‚Äîdibuka untuk komunitas internet secara luas. Ini memindahkan Web dari eksperimen internal menjadi infrastruktur informasi terbuka.

Maknanya terletak pada aksesibilitas universal: siapa pun dapat menjalankan server, menulis halaman HTML, dan menautkannya, sehingga jaringan pengetahuan kolektif bisa tumbuh secara desentralisasi.

Pada 1991, Tim Berners-Lee dan tim di CERN membuka akses World Wide Web ke komunitas internet global. Server pertama (info.cern.ch) dan perangkat lunak pendukung seperti WorldWideWeb (kemudian Nexus) memungkinkan siapa pun menjelajah dan menerbitkan dokumen terhubung melalui URL, HTTP, dan HTML. Pengumuman dilakukan di newsgroup alt.hypertext, mengundang komunitas untuk mencoba dan mengembangkan teknologi baru ini.

Inilah momen ketika Web beralih dari eksperimen internal menjadi infrastruktur publik yang siap diadopsi luas.

Latar Belakang & Teknologi Awal

  • Server web httpd berjalan di mesin NeXT di CERN.
  • Browser/editor Nexus mendukung penulisan dan penelusuran dokumen.
  • Hypertext memungkinkan navigasi bebas antardokumen lewat tautan.
  • URL/URI menyediakan cara universal untuk mengalamatkan sumber daya.
  • HTTP 0.9 sebagai protokol awal yang sangat sederhana (request satu baris, response dokumen).

Dampak

Perilisan publik mempercepat kolaborasi riset, memicu munculnya server dan situs baru di universitas hingga industri, dan menjadi fondasi ekonomi digital di dekade berikutnya. Prinsip open yang diusung (tanpa royalti) mempercepat adopsi dan inovasi.

Perbandingan dengan Alternatif Saat Itu

  • Gopher: Terstruktur hierarkis, mudah ditelusuri, tetapi kurang fleksibel dibanding tautan antar dokumen.
  • FTP: Unggul untuk transfer file, namun tidak cocok sebagai medium publishing hypertext.
  • Usenet/WAIS: Fokus pada diskusi dan pencarian, bukan dokumen tertaut yang universal.

Garis Waktu Singkat

  • 1989: Proposal awal Web di CERN.
  • 1990: Prototipe pertama: HTTP, HTML, browser/editor WorldWideWeb, dan server httpd.
  • 1991: Web diumumkan ke publik; server info.cern.ch online.
  • 1992: Komunitas luar CERN mulai menjalankan server dan menulis browser.

Glosarium Singkat

  • HTTP 0.9: Versi awal HTTP, hanya mendukung GET dan mengirimkan dokumen HTML polos.
  • Nexus: Nama baru untuk browser/editor WorldWideWeb buatan Berners-Lee.

Studi Kasus: info.cern.ch

  • Konten awal: Dokumentasi proyek Web, cara menjalankan server, dan contoh dokumen.
  • Tujuan: Menjadi pusat rujukan agar pihak lain bisa mereplikasi dan memperluas jaringan Web.
  • Dampak: Menjadi model halaman dokumentasi teknis yang ringkas namun fungsional.

Praktik Terbaik (pada masanya)

  • Gunakan hyperlink yang jelas dan deskriptif untuk memandu navigasi.
  • Sederhanakan struktur dokumen agar bisa di-render oleh beragam browser awal.
  • Publikasikan dokumentasi teknis agar komunitas dapat berkontribusi.

Miskonsepsi Umum

  • Web = Internet: Web hanyalah salah satu layanan di atas Internet, bukan keseluruhannya.
  • Web langsung populer: Adopsi membutuhkan waktu; lonjakan besar terjadi setelah browser grafis populer.

Dampak Jangka Panjang

Model dokumen tertaut mendorong pengetahuan terdistribusi, mendorong open access, dan mempercepat inovasi lintas disiplin. Prinsip desain Web (kesederhanaan, universalitas, desentralisasi) tetap relevan untuk teknologi modern.

Garis Waktu Singkat

  • 1991: Web diumumkan ke publik; server dan browser tersedia.
  • 1992: Pertumbuhan server web di luar CERN.
  • 1993: Mosaic memopulerkan web grafis.
