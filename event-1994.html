  • Pendiri: Tim Berners-Lee
  • Lokasi awal: MIT (AS), kemudian INRIA/Keio
  • Misi: Mengembangkan standar web terbuka

Untuk memastikan Web tetap terbuka, interoperabel, dan dapat diakses, Tim Berners-Lee mendirikan W3C pada 1994. Konsorsium ini menyatukan akademisi dan industri untuk menyusun spesifikasi seperti HTML, CSS, DOM, XML, dan banyak lagi.

Standar W3C memungkinkan berbagai browser, perangkat, dan alat bekerja dengan cara yang konsisten sehingga ekosistem web tumbuh cepat.

Bidang Kerja Utama

  • Spesifikasi inti: HTML, CSS, DOM, SVG.
  • Aksesibilitas: WCAG untuk membuat web inklusif.
  • Internasionalisasi: Dukungan bahasa dan penulisan non-Latin.

Dampak

Dengan proses standar yang terbuka, inovasi dapat terjadi tanpa hambatan vendor lock-in. Developer memiliki rujukan resmi, dan pengguna menikmati pengalaman yang lebih seragam lintas platform.
