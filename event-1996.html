Pengertian

CSS Level 1 adalah standar styling yang memisahkan tampilan dari struktur HTML, sehingga desain dapat diubah tanpa menyentuh markup. CSS memperkenalkan model deklaratif berbasis selektor dengan mekanisme cascade dan inheritance.

Intinya, CSS memungkinkan konsistensi visual lintas halaman, meningkatkan pemeliharaan, dan mendukung aksesibilitas karena HTML dapat fokus pada semantik.

Di awal era web, styling seperti warna, font, dan tata letak sering ditulis langsung di HTML. Setiap elemen punya atribut style sendiri-sendiri. Praktik ini membuat kode jadi berantakan, sulit dirawat, dan tidak bisa diubah dengan mudah. CSS Level 1 hadir sebagai solusi dengan memindahkan semua aturan presentasi ke file terpisah. Sekarang, satu perubahan bisa berdampak ke seluruh situs.

Inti dari CSS adalah sifatnya yang deklaratif, berbasis selektor, dan menggunakan mekanisme cascading serta inheritance untuk menyelesaikan konflik gaya secara konsisten.

Fitur Kunci CSS Level 1

  • Selector dasar: Bisa memilih elemen berdasarkan nama tag, class, atau ID.
  • Typography: Kontrol lengkap untuk keluarga font, ukuran, tebal, miring, dan spasi huruf maupun baris.
  • Warna & latar: Bisa mengatur warna teks, warna latar, dan gambar latar.
  • Box model: Kontrol margin, border, padding, dan ukuran elemen.
  • Units: Berbagai satuan seperti px, em, dan % untuk membuat desain yang lebih responsif.
  • Alignment: Mengatur perataan teks dan elemen dengan text-align dan vertical-align.
  • Specificity: Sistem prioritas untuk menentukan aturan mana yang akan diterapkan saat ada konflik.

Pemisahan Presentasi dari Struktur

Dengan CSS, HTML fokus pada makna dan struktur dokumen (judul, paragraf, daftar), sementara CSS mengatur tampilannya. Pendekatan ini punya beberapa keuntungan:

  • Fleksibilitas desain: Bisa mengubah tema atau warna seluruh situs hanya dari satu file CSS.
  • Aksesibilitas: Struktur HTML yang bersih memudahkan pembaca layar untuk mengakses konten.
  • Ukuran dokumen: HTML jadi lebih ringan dan mudah dirawat karena tidak ada tag styling di dalamnya.

Tantangan Kompatibilitas Awal

Implementasi CSS di awal berbeda-beda antara browser. Pengembang harus melakukan banyak trik dan pengujian lintas-browser untuk memastikan situs terlihat benar di semua browser. Standarisasi oleh W3C menjadi kunci untuk menyatukan perilaku CSS di berbagai browser.

Garis Waktu Singkat

  • 1994: W3C didirikan untuk menstandarkan teknologi web.
  • 1996: CSS Level 1 dipublikasikan sebagai rekomendasi resmi.
  • 1998: CSS Level 2 menambah fitur lanjutan seperti positioning, media queries, dan lain-lain.

Glosarium Singkat

  • Cascade: Mekanisme penggabungan aturan CSS saat terjadi konflik berdasarkan urutan dan spesifisitas.
  • Inheritance: Properti tertentu diwariskan dari elemen induk ke elemen anak secara otomatis.
  • Box model: Cara elemen dihitung ukurannya, terdiri dari content, padding, border, dan margin.
  • Specificity: Nilai prioritas selektor untuk menentukan aturan mana yang menang saat ada konflik.
