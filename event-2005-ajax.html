  • Istilah: Asynchronous JavaScript and XML (Ajax)
  • Populer oleh: Jesse James Garrett
  • Contoh awal: Google Maps, Gmail

Ajax memisahkan interaksi UI dari pemuatan halaman penuh dengan melakukan permintaan data secara asinkron. Pengguna merasakan aplikasi web yang lebih cepat dan mulus.

Ajax membuka jalan bagi arsitektur single-page application dan pola API JSON modern.

Teknologi Kunci

  • XMLHttpRequest (XHR) untuk permintaan HTTP asinkron.
  • JSON menggantikan XML sebagai format pertukaran data yang ringan.
  • Manipulasi DOM untuk memperbarui tampilan tanpa reload.

Contoh Ikonik

  • Google Maps: Panning/zooming mulus tanpa muat ulang.
  • Gmail: Email berbasis web dengan interaksi cepat.
