Pengertian

Web 2.0 adalah fase evolusi web yang menekankan partisipasi pengguna dalam menciptakan dan membagikan konten, didukung oleh jejaring sosial dan platform kolaboratif. Ini mengubah web dari sekadar media publikasi menjadi ruang interaksi.

Dalam praktiknya, API terbuka, tagging, dan umpan informasi memungkinkan integrasi antar layanan dan pengalaman pengguna yang lebih kaya.

Web 2.0 menandai pergeseran dari halaman statis menjadi platform partisipatif. Pengguna tidak hanya mengonsumsi, tetapi juga menciptakan dan berbagi konten secara masif.

API terbuka, tagging, dan umpan (RSS/Atom) mendorong integrasi lintas layanan dan pertumbuhan ekosistem.

Ciri & Teknologi Pendukung

  • Jejaring sosial dan platform UGC (user-generated content).
  • Ajax untuk interaksi lebih mulus tanpa reload penuh.
  • API dan mashup yang memungkinkan integrasi antar layanan.

Dampak Sosial & Ekonomi

Model bisnis baru (iklan bertarget, ekonomi kreator) dan budaya berbagi berkembang pesat. Kolaborasi skala besar seperti Wikipedia membuktikan kekuatan kontribusi komunitas.
