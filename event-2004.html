Web 2.0 menandai pergeseran dari halaman statis menjadi platform partisipatif. Pengguna tidak hanya mengonsumsi, tetapi juga menciptakan dan berbagi konten secara masif.

API terbuka, tagging, dan umpan (RSS/Atom) mendorong integrasi lintas layanan dan pertumbuhan ekosistem.
