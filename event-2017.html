  • Organisasi: W3C
  • Status: CSS Grid Layout Module Level 1
  • Dukungan browser: 2017 (Chrome, Firefox, Safari)
  • Jenis: Sistem tata letak 2D

Meskipun Flexbox sudah membantu banyak hal, ia hanya bisa mengatur tata letak satu dimensi (baris atau kolom). Untuk layout kompleks seperti dashboard atau majalah, pengembang masih harus menggunakan kombinasi float, position, dan trik CSS lainnya yang rumit. CSS Grid Layout hadir sebagai solusi yang dirancang khusus untuk tata letak dua dimensi.

Grid memungkinkan pengembang mendefinisikan dan mengontrol baris dan kolom secara bersamaan. Ini adalah pertama kalinya CSS punya sistem layout native yang powerful untuk tata letak kompleks tanpa perlu trik atau hack.

Latar Belakang: Sebelum Grid

Sebelum Grid muncul, pengembang mengandalkan float, position, dan table untuk membuat layout. Float dirancang untuk teks mengelilingi gambar, bukan untuk layout. Position juga tidak ideal untuk struktur kompleks. Flexbox membantu untuk tata letak satu dimensi, tapi untuk struktur kompleks seperti dashboard atau layout majalah, tetap butuh kombinasi teknik yang rumit.

Fitur Kunci CSS Grid

  • Grid container & items: Elemen induk menjadi grid, sementara anak-anaknya menjadi items yang bisa ditempatkan di mana saja.
  • Explicit & implicit tracks: Baris dan kolom bisa didefinisikan eksplisit atau dibuat otomatis oleh browser.
  • Grid lines & areas: Kontrol presisi posisi item melalui garis atau area bernama yang lebih mudah dibaca.
  • fr unit: Unit fraksi untuk distribusi ruang yang fleksibel, lebih intuitif daripada persentase.
  • Auto-placement: Grid secara otomatis menempatkan item yang tidak punya posisi eksplisit.
  • Alignment: Kontrol perataan konten di dalam grid cells dengan justify-items, align-items, dan lainnya.

Grid vs. Flexbox: Kapan Menggunakan?

  • Grid: Ideal untuk layout dua dimensi (baris + kolom), struktur kompleks seperti dashboard, dan layout majalah.
  • Flexbox: Ideal untuk layout satu dimensi, alignment konten dalam satu baris atau kolom, dan distribusi ruang yang fleksibel.
  • Kombinasi: Grid untuk struktur utama halaman, Flexbox untuk komponen internal seperti navbar atau card.

Dampak pada Industri

  • Pengurangan ketergantungan pada framework CSS kompleks untuk layout dasar, karena Grid sudah powerful sendiri.
  • Kemampuan membuat layout majalah dan dashboard yang sebelumnya sulit atau mustahil dengan teknik lama.
  • Peningkatan produktivitas pengembang: kode lebih bersih, mudah dirawat, dan lebih mudah dipahami.
  • Evolusi tool pengembangan: browser DevTools dengan Grid visualizer yang membantu debugging dan pembelajaran.

Perbandingan dengan Teknologi Sebelumnya

  • Vs. Float: Grid tidak memerlukan clearing, lebih prediktif, dan fleksibel. Tidak perlu trik seperti clearfix.
  • Vs. Table layout: Grid menggunakan CSS modern tanpa markup semantik yang salah. Tabel tetap untuk data, bukan layout.
  • Vs. Framework grid: Native CSS, tidak perlu library eksternal, lebih ringan, dan performant.

Glosarium Singkat

  • Grid container: Elemen yang menerapkan display: grid.
  • Grid item: Anak langsung dari grid container yang ditempatkan di dalam grid.
  • Grid line: Garis pembagi (vertikal atau horizontal) yang membentuk grid.
  • Grid track: Ruang antara dua grid lines, bisa berupa baris atau kolom.
  • Grid cell: Area terkecil di grid, intersection dari satu baris dan satu kolom.
  • Grid area: Satu atau lebih grid cells yang membentuk area persegi panjang.
  • fr unit: Unit fraksi yang membagi ruang tersisa secara proporsional, lebih intuitif daripada persentase.

Fakta Menarik

  • Microsoft IE10: Browser pertama yang mengimplementasikan Grid, tapi dengan syntax berbeda yang tidak populer. Standar W3C kemudian berbeda dan lebih baik.
  • Rachel Andrew: Salah satu pionir yang mempopulerkan Grid melalui artikel, tutorial, dan konferensi. Ia membantu banyak pengembang memahami Grid.
  • Subgrid: Fitur lanjutan yang memungkinkan grid item punya grid sendiri yang mengikuti tracks parent grid. Masih dalam pengembangan dan belum didukung semua browser.
  • Dukungan universal: Grid sekarang didukung di semua browser modern, termasuk mobile, membuatnya aman untuk digunakan di produksi tanpa khawatir kompatibilitas.
