Tahun 1989 menjadi titik awal revolusi digital ketika fisikawan Inggris Tim Berners-Lee mengusulkan ide yang sederhana namun revolusioner. Di CERN, tempat berbagai sistem komputer terpisah-pisah, ia melihat kebutuhan untuk menghubungkan informasi agar mudah diakses. Gagasannya adalah menggabungkan konsep hypertext dengan jaringan internet yang sudah ada.

Visi Berners-Lee adalah menciptakan "web" dokumen yang saling tertaut, bisa diakses siapa saja, dan mudah dinavigasi. Dari visi ini lahir tiga pilar utama: URL, HTTP, dan HTML.

Konsep Hypertext

Hypertext memungkinkan dokumen saling terhubung melalui tautan (link). Pengguna bisa melompat dari satu konteks ke konteks lain tanpa harus mengikuti urutan linear. Konsep ini menjadi dasar cara kita menjelajah web sampai sekarang.

Teknologi Kunci yang Diperkenalkan

  • HTML (HyperText Markup Language): Bahasa sederhana untuk menyusun struktur dokumen seperti judul, paragraf, tautan, dan daftar.
  • URL/URI: Sistem penamaan universal untuk menemukan sumber daya di jaringan, atau yang kita kenal sebagai alamat web.
  • HTTP (HyperText Transfer Protocol): Protokol ringan untuk meminta dan mengirim dokumen hypertext antara browser dan server.

Dampak dan Signifikansi

Ketiga teknologi ini membentuk fondasi yang terbuka dan bisa diperluas. Siapa pun bisa membuat halaman, menautkannya dengan halaman lain, dan membagikannya ke seluruh dunia. Sejak dirilis ke publik pada 1991, web mengubah cara kita berkomunikasi, belajar, berbisnis, dan mengakses informasi.

Garis Waktu Singkat

  • 1989: Proposal awal Web di CERN.
  • 1990: Prototipe pertama HTTP, HTML, dan browser WorldWideWeb dibuat.
  • 1991: Web dirilis untuk publik di internet.
  • 1993: Mosaic memperkenalkan penelusuran grafis yang lebih ramah pengguna.

Latar Belakang Teknis dan Sosial

Sebelum Web muncul, berbagi informasi riset dilakukan lewat FTP, email, dan usenet. Sistem-sistem ini fokus pada file atau pesan, bukan dokumen yang saling tertaut. Data tersebar di berbagai sistem operasi dan format, sehingga sulit ditelusuri. CERN sebagai organisasi riset kolaboratif butuh cara yang luwes, murah, dan lintas platform untuk publikasi internal dan eksternal.

Perbandingan dengan Alternatif Saat Itu

  • Gopher: Menyajikan menu hierarkis yang terstruktur, tapi kurang fleksibel dibanding tautan bebas antar dokumen pada Web.
  • FTP: Efektif untuk transfer file, tapi tidak menyediakan navigasi dan presentasi dokumen yang mudah.
  • WAIS/Usenet: Mendukung pencarian dan diskusi, tapi tidak menawarkan pemodelan dokumen tertaut yang universal.

Prinsip Desain yang Diutamakan

  • Kesederhanaan: Spesifikasi dibuat minimal agar mudah diimplementasikan dan diadopsi banyak orang.
  • Universalitas: Tidak bergantung pada perangkat keras, sistem operasi, atau bahasa pemrograman tertentu.
  • Desentralisasi: Siapa pun bisa menerbitkan konten tanpa perlu izin dari otoritas pusat.
  • Ekstensibilitas: Format dan protokol bisa berkembang tanpa memutus kompatibilitas dengan versi sebelumnya.

Tonggak Lanjutan

  • 1993: CERN merilis teknologi Web bebas royalti, mendorong adopsi luas.
  • 1994: World Wide Web Consortium (W3C) dibentuk untuk menstandarkan teknologi Web.
  • 1994‚Äì1995: Munculnya browser komersial dan lonjakan situs web publik.

Dampak Jangka Panjang

Web memicu lahirnya ekosistem baru: mesin pencari, perdagangan elektronik, media sosial, komputasi awan, hingga aplikasi kaya di browser. Sifatnya yang terbuka memungkinkan inovasi cepat dan kolaborasi global lintas disiplin ilmu.

Glosarium Singkat

  • Hypertext: Teks dengan tautan yang menghubungkan ke dokumen lain.
  • URL: Alamat untuk mengidentifikasi lokasi sumber daya di Web.
  • HTTP: Protokol pertukaran data antara browser dan server.
  • HTML: Bahasa untuk menyusun struktur halaman web.

Fakta Menarik

Berners-Lee sengaja tidak mematenkan Web agar tetap bebas dan terbuka. Keputusan ini mempercepat adopsi global dan mendorong inovasi tanpa hambatan lisensi. Jika Web dipatenkan, mungkin kita tidak akan melihat perkembangan secepat ini.
