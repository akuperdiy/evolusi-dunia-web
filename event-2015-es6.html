  • Standar: ECMA-262

ES6 menghadirkan fitur bahasa modern yang menyederhanakan penulisan kode, modularisasi, dan manajemen asinkron. Ini menjadi baseline baru untuk pengembangan JavaScript modern di browser dan server. Dukungan bertahap di browser dibantu oleh transpiler seperti Babel.

Fitur Penting

  • Modules (import/export) untuk modularisasi kode.
  • Classes mempermudah pola OOP.
  • let/const untuk scoping yang lebih aman.
  • Arrow functions dengan lexical this.
  • Promises untuk asinkron yang lebih bersih.
  • Destructuring, rest/spread, template literals untuk ergonomi penulisan.

Dampak

  • Standarisasi fitur meminimalkan kebutuhan polyfill khusus.
  • Ekosistem tooling (Babel/Webpack/Rollup) berkembang pesat.
